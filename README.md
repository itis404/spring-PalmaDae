# Функционал

Сайт позволяет создавать пользователей через почту(и её подтверждение), также предусмотрен вход через аккаунт сервиса Яндекс

Возможность добавлять донации разных типов(Цельная кровь, плазма, эритроциты, тромбоциты)

Создавать мероприятия для доноров

Админ панель, для подтверждения донаций и мероприятий

Также сайт показывает информацию о пунктах сдачи крови в городе пользователя)

# Android
